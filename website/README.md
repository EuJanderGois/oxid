# Oxid Documentation

This folder contains the Docusaurus site for Oxid.

## Commands

```bash
npm install
npm run start
npm run typecheck
npm run build
```

## Structure

- `docs/` for product and API documentation
- `blog/` reserved for future posts and release notes
- `i18n/` reserved for future translations
- `src/` for the homepage and custom theme code
- `static/` for public assets

## Content rules

- keep documentation aligned with the current engine and CLI
- use English slugs for stable routes
- document shipped behavior first
- move future ideas into the roadmap instead of presenting them as ready
